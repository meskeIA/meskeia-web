❓ Preguntas Frecuentes

¿Mis datos son privados?

Sí, completamente. Todos tus datos se guardan únicamente en tu navegador usando localStorage. No se envía ninguna información a servidores externos. Tienes control total de tu información financiera.

¿Puedo usar la app sin conexión a internet?

Sí, funciona 100% offline. No requiere conexión a internet para registrar gastos, ver tu balance o consultar el historial.

¿Cómo categorizo mis gastos?

Al añadir un gasto, seleccionas una categoría de la lista predefinida: Vivienda, Alimentación, Transporte, Salud, Ocio, Ropa, Suscripciones u Otros. Esto te permite analizar en qué gastas más.

¿Puedo exportar mis datos?

Sí, tienes 2 opciones: (1) Exportar el mes actual a CSV para Excel/Google Sheets, (2) Exportar TODO a JSON para backup completo de todos los meses. También puedes importar backups JSON para restaurar tus datos.

¿Cómo hago backup de mis datos?

Usa el botón 'Exportar TODO a JSON' para descargar un backup completo con todos los meses y transacciones. Guarda este archivo en tu nube (Google Drive, Dropbox) y podrás restaurarlo con 'Importar desde JSON'.
