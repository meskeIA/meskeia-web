📖 Guía del Cuaderno Digital

Aprende a aprovechar al máximo tu organizador personal con Markdown

🚀 Bienvenido a tu Cuaderno Digital

El Cuaderno Digital es tu herramienta personal para organizar notas, ideas, proyectos y documentos. Todo se guarda de forma 100% privada en tu navegador, sin enviar datos a servidores externos.

🔒 Privacidad Total: Todos tus documentos se almacenan únicamente en tu navegador. Nadie más puede acceder a ellos, ni siquiera nosotros.

✨ Primeros Pasos

1Crear tu Primera Nota

  • Haz clic en "✏️ Nuevo Documento"
  • Escribe un título descriptivo para tu nota
  • Selecciona una categoría (o crea una nueva)
  • Elige un color para identificar rápidamente tu nota
  • Redacta tu contenido usando Markdown en el panel izquierdo
  • Ve el resultado en tiempo real en el panel derecho

2Guardar y Organizar

  • Haz clic en "Guardar Documento" cuando termines
  • Tu nota aparecerá como una tarjeta colorida en la pantalla principal
  • Usa las categorías para agrupar notas relacionadas
  • El buscador te ayuda a encontrar cualquier nota rápidamente

📝 Guía Rápida de Markdown

Formato de Texto:

**Texto en negrita** *Texto en cursiva* `Código inline` ~~Texto tachado~~

Títulos y Estructura:

# Título Principal (H1) ## Subtítulo (H2) ### Título Menor (H3)

Listas:

- Lista con viñetas - Otro elemento - Tercer elemento 1. Lista numerada 2. Segundo elemento 3. Tercer elemento - [x] Tarea completada - [ ] Tarea pendiente

Enlaces y Código:

[Texto del enlace](https://ejemplo.com) ```javascript function ejemplo() { console.log("Bloque de código"); } ```

Citas y Separadores:

> Esto es una cita importante > que puede ocupar varias líneas --- (Línea horizontal para separar secciones)

📋 Gestión de tus Notas

Operaciones con Tarjetas:

  • 👁️ Ver: Haz clic en cualquier tarjeta para leer el contenido completo
  • ✏️ Editar: Hover sobre la tarjeta y haz clic en el icono de editar
  • 🗑️ Eliminar: Hover sobre la tarjeta y haz clic en el icono de eliminar
  • 🔍 Buscar: Usa la barra de búsqueda para encontrar notas específicas
  • 🏷️ Filtrar: Selecciona una categoría para ver solo notas de ese tipo

Funciones del Lector:

  • ✏️ Editar: Modifica la nota directamente desde el lector
  • 🖨️ Imprimir: Genera una versión limpia para imprimir
  • 💾 Exportar: Descarga la nota como archivo HTML

💾 Backup y Seguridad

⚠️ Importante: Tus notas se guardan en el navegador. Si borras los datos del navegador o cambias de dispositivo, perderás tus notas.

Estrategia de Backup Recomendada:

  1. Backup Regular: Usa "💾 Exportar Todo" cada semana o mes
  2. Archivo Seguro: Guarda el archivo JSON en múltiples ubicaciones
  3. Sincronización: Sube el backup a tu nube personal (Drive, Dropbox, etc.)
  4. Restauración: Usa "📂 Importar Backup" para recuperar tus notas

Migración Entre Dispositivos:

  1. En el dispositivo origen: "💾 Exportar Todo"
  2. Transfiere el archivo JSON al nuevo dispositivo
  3. En el nuevo dispositivo: "📂 Importar Backup"
  4. Elige si combinar o reemplazar las notas existentes
✅ Consejo Pro: Añade el Cuaderno Digital a los marcadores de tu navegador para acceso rápido desde cualquier pestaña.

⚡ Acceso Rápido y Productividad

Añadir a Marcadores:

  1. Con el Cuaderno Digital abierto, presiona Ctrl+D (Windows/Linux) o Cmd+D (Mac)
  2. Guárdalo en la barra de marcadores para acceso con un clic
  3. Puedes renombrarlo como "📝 Cuaderno" para identificarlo fácilmente

Atajos de Teclado en el Editor:

  • Ctrl+S - Guardar documento
  • Ctrl+Z - Deshacer
  • Ctrl+Y - Rehacer
  • Tab - Indentar texto
  • Escape - Cerrar modal

Consejos de Organización:

  • 🎨 Código de Colores: Asigna colores por urgencia (rojo=urgente, azul=información)
  • 📅 Prefijos de Fecha: Usa "2025-01-19" al inicio de títulos para orden cronológico
  • 🏷️ Convención de Categorías: Mantén nombres cortos y consistentes
  • 🔍 Palabras Clave: Incluye términos clave en el contenido para búsquedas eficientes

🔧 Solución de Problemas

Problemas Comunes:

❓ No veo mis notas después de cerrar el navegador
Las notas están guardadas en el navegador específico. Verifica que estás usando el mismo navegador y no has borrado los datos de navegación.
❓ El backup no se puede importar
Asegúrate de que el archivo es un JSON válido exportado desde el Cuaderno Digital. Verifica que no se haya corrompido durante la transferencia.
❓ La vista previa de Markdown no funciona
Refresca la página y vuelve a abrir el editor. Si persiste, verifica que no hay caracteres especiales que puedan causar conflictos.
❓ No puedo crear nuevas categorías
Simplemente escribe el nombre de la nueva categoría en el campo. El sistema la creará automáticamente al guardar.
