💡 ¿Qué es un Hash?

  • ✓ Un hash es una cadena única generada desde datos
  • ✓ Cualquier cambio en los datos produce un hash diferente
  • ✓ Es imposible obtener los datos originales desde el hash
  • ✓ Se usa para verificar integridad y autenticidad
  • ✓ Cada algoritmo tiene diferente nivel de seguridad

🎯 Casos de Uso

  • ✓ Verificar descargas: Compara hash del archivo descargado
  • ✓ Identificadores únicos: Genera IDs para bases de datos
  • ✓ Detectar duplicados: Compara archivos por su hash
  • ✓ Integridad de datos: Verifica que no se alteraron
  • ✓ Firmas digitales: Base para certificados

🔒 Algoritmos Disponibles

  • MD5: 128 bits - Rápido, no seguro para criptografía
  • SHA-1: 160 bits - Obsoleto para seguridad
  • SHA-256: 256 bits - Recomendado para seguridad
  • SHA-384: 384 bits - Alta seguridad
  • SHA-512: 512 bits - Máxima seguridad
