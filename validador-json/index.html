💡 ¿Cómo usar esta herramienta?

  • Pega o escribe tu código JSON/XML en el área de entrada
  • Haz clic en "Validar" para verificar si el código es válido
  • Usa "Formatear" para obtener una versión legible con indentación
  • Usa "Minificar" para comprimir el código (elimina espacios y saltos de línea)
  • Copia el resultado formateado directamente al portapapeles
